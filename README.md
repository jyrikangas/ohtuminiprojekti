# Ohjelman käyttö

  1. Kopioi repositorio omalle koneellesi
  2. Asenna riippuvuudet komennolla:
  ```bash
  poetry install
  ``` 
  4. Siirry kansioon src
  3. Siirry virtuaaliympäristöön komennolla:
  ```bash
  poetry shell
  ```
  4. Käynnistä sovellus komennolla:
  ```bash
  flask run
  ```
